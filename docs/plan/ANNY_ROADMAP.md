# ANNY Roadmap - Mission Zero and Mission One

Document version: v1.1

ANNY is an AI Trading Companion platform built around transparent results and
private methods. It does not guarantee returns and does not hold user funds.

## Mission Zero

Goal: validate the product with an invite-only founder community before a model
marketplace.

- Target 32 active Founding Captains; maximum 64.
- Demo trading, approved-user trading gates, AI Companion, and Flight Recorder.
- Record entry, exit, risk, result, and execution history.
- Success criteria: 32 active users, 80% weekly retention, 100 completed
  missions, stable execution, and useful community feedback.

Current model milestone:

- [x] Register ANNY Basic v1.2 as the first versioned model policy.
- [x] Encode entry, no-trade, phase, stop, leverage-cap, and rescue rules in Go.
- [x] Preserve illustrative win/loss scenarios as non-performance evidence.
- [x] Implement canonical CDC/QQE indicator calculations and dual-timeframe adapter.
- [ ] Complete fee-adjusted paper and walk-forward validation.
- [ ] Connect model decisions to Mission confirmation and Flight Recorder on testnet.
- [ ] Complete restart-safe timed Mission jobs before production execution.

## Mission One

Goal: launch the first private AI Trading Model Marketplace.

- Creators publish executable models; source, prompts, parameters, weights, and
  internal logic remain private.
- Users receive model execution through ANNY's confirmation and risk controls.
- Verification means operational and transparency requirements passed. It is
  not a profitability guarantee or financial advice.
- Promotion affects visibility only, never verification or organic ranking.
- Marketplace subscriptions and revenue share require legal review before launch.

## Later Missions

- Mission Two: public marketplace, verified models, and revenue sharing.
- Mission Three: execution-hash anchoring and model reputation.
- Mission Four: multiple exchanges, enterprise deployments, and developer API.
